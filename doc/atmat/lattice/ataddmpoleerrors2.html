ataddmpoleerrors2

PURPOSE

ataddmpoleerrors2 adds multipole errors to certain elements of ring and

SYNOPSIS

function ring = ataddmpoleerrors2(ring,elements,type,newindex,strength,radius)

DESCRIPTION 

ataddmpoleerrors2 adds multipole errors to certain elements of ring and
outputs a new ring

[newring] = ATADDMPOLEERROS(ring,elements,type,newindex,strength,radius)

ring = input ring
elements can either be "quadrupole", "sextupole" or an array of indices.  
 It is assumed that they are all of the same type
newindex: index of Multipole to add
strength: strength is either a fixed value (B,A,R) to add systematic multipole
errors or else a field error structure with arbitrary predetermined values.  
radius: the reference radius for defining the multipole error.

 The formula for the added errors is
 B^(N)_(n) = radius^(n-N)*b_n/b_N
 It represents the relative field error to the design value at the ref.
 radius
For example, to add a random octupole error of 1e-4 at 25 mm, relative to all
quadrupoles:
 newring =ataddrandmpole(ring,'quadrupole',4,.1e-4,.025,1);

See also: ataddmpolecomppoly attiltelem atshiftelem
